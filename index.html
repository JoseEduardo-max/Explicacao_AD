Árvore de Decisão

Como o modelo pensa e toma decisões

O que ela faz de verdade?

  • Divide os dados em grupos
  • Busca deixar os grupos mais puros
  • Continua até chegar em uma decisão

Ideia: separar dados parecidos

Impureza

  • Puro → uma única classe
  • Impuro → mistura de classes

Objetivo: diminuir a impureza

Exemplo

Grupo   Classe
A       Cachorro
B       Cachorro
C       Gato

→ Grupo impuro

Como a árvore divide?

  • Idade > 18?
  • Altura < 1.70?
  • Tem pelos?

→ Cada pergunta cria uma divisão

Como escolhe a melhor divisão?

  1. Testa várias perguntas
  2. Mede a impureza
  3. Escolhe a melhor

Gini

Gini = 1 - Σ p²

  • 0 = perfeito
  • 0.5 = bagunçado

Entropia

Entropy = - Σ p log₂(p)

  • 0 = organizado
  • Alto = bagunça

Ganho de Informação

Quanto uma divisão melhora os dados

→ Escolhe a maior redução de impureza

Como a árvore cresce

  1. Começa com todos os dados
  2. Escolhe melhor divisão
  3. Divide
  4. Repete

Quando ela para?

  • Nó puro
  • Limite de profundidade
  • Poucos dados

Parâmetros

  • criterion: gini ou entropy
  • max_depth: profundidade
  • min_samples_split: mínimo para dividir
  • min_samples_leaf: mínimo na folha

Overfitting

  • Modelo decora os dados
  • Erra dados novos

Solução: limitar a árvore

Underfitting

  • Modelo muito simples
  • Não aprende direito

Comparação com KNN

  • KNN → usa distância
  • Decision Tree → cria regras

Por que usar?

  • Fácil de entender
  • Rápido
  • Interpretável

Resumo

A árvore aprende fazendo perguntas e dividindo os dados até encontrar grupos cada vez mais organizados.
