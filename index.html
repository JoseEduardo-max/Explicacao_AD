🌳 Árvore de Decisão

Como o modelo pensa e toma decisões

🧠 O que ela faz?

  • Divide os dados em grupos
  • Deixa os grupos mais puros
  • Continua até decidir

Separar dados parecidos

📦 Impureza

Puro → uma única classe

Impuro → mistura de classes

Objetivo: diminuir a impureza

🐶 Exemplo

A → Cachorro
B → Cachorro
C → Gato

Grupo impuro

✂️ Como divide?

  • Idade > 18?
  • Altura < 1.70?
  • Tem pelos?

🎯 Melhor divisão

  1. Testa perguntas
  2. Mede impureza
  3. Escolhe a melhor

🧮 Gini

Gini = 1 - Σ p²

✔ 0 = perfeito

⚠ 0.5 = bagunçado

🔥 Entropia

Entropy = - Σ p log₂(p)

✔ 0 = organizado

⚠ Alto = bagunça

⚖️ Ganho de Informação

Quanto a divisão melhora os dados

Escolhe a maior redução de impureza

🌳 Crescimento

  1. Começa com todos os dados
  2. Escolhe melhor divisão
  3. Divide
  4. Repete

🧠 Quando para?

  • Nó puro
  • Limite de profundidade
  • Poucos dados

⚙️ Parâmetros

criterion: gini / entropy

max_depth: profundidade

min_samples_split: mínimo para dividir

min_samples_leaf: mínimo na folha

⚠️ Overfitting

Modelo decora os dados

Erra dados novos

Solução: limitar a árvore

📉 Underfitting

Modelo simples demais

Não aprende direito

📊 KNN vs Decision Tree

KNN → usa distância

Decision Tree → cria regras

🚀 Por que usar?

  • Fácil de entender
  • Rápido
  • Interpretável

🎤 Resumo

A árvore aprende fazendo perguntas e dividindo os dados até deixá-los organizados.
