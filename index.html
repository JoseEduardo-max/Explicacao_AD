🌳 Árvore de Decisão

Como o modelo pensa e toma decisões

“Bom, hoje eu vou explicar como funciona a árvore de decisão, que é um modelo de machine learning que aprende a tomar decisões fazendo perguntas.” “Basicamente, ela pensa de forma parecida com a gente: fazendo perguntas simples até chegar numa conclusão.”

🧠 O que ela faz?

  • Divide os dados em grupos
  • Deixa os grupos mais puros
  • Continua até decidir

Separar dados parecidos

“A árvore pega um conjunto de dados e começa a dividir em grupos menores.” “O objetivo dela é deixar esses grupos o mais organizados possível, ou seja, com dados parecidos entre si.” “No final, ela consegue separar e classificar os dados corretamente.”

📦 Impureza

Puro → uma única classe

Impuro → mistura de classes

Objetivo: diminuir a impureza

“Aqui entra o conceito de impureza.” “Um grupo puro é quando todos os dados são da mesma classe.” “Um grupo impuro é quando tem mistura.” “A árvore sempre tenta diminuir essa impureza.”

🐶 Exemplo

A → Cachorro
B → Cachorro
C → Gato

Grupo impuro

“Por exemplo, aqui temos dois cachorros e um gato.” “Isso é um grupo impuro.” “A árvore vai tentar separar isso, criando perguntas que dividam melhor esses dados.”

✂️ Como divide?

  • Idade > 18?
  • Altura < 1.70?
  • Tem pelos?
“A árvore faz perguntas como: idade, altura ou alguma característica.” “Cada pergunta cria uma divisão.” “É como se ela estivesse cortando os dados em partes menores.”

🎯 Melhor divisão

  1. Testa perguntas
  2. Mede impureza
  3. Escolhe a melhor
“Mas ela não escolhe qualquer pergunta.” “Ela testa várias possibilidades, mede qual deixa os dados mais organizados…” “E escolhe a melhor divisão.”

🧮 Gini

Gini = 1 - Σ p²

✔ 0 = perfeito

⚠ 0.5 = bagunçado

“O Gini é uma forma de medir a bagunça dos dados.” “Se for zero, está perfeito — só tem uma classe.” “Quanto maior, mais misturado está.”

🔥 Entropia

Entropy = - Σ p log₂(p)

✔ 0 = organizado

⚠ Alto = bagunça

“A entropia é parecida com o Gini, também mede desorganização.” “Quanto maior o valor, mais bagunça.” “Quanto menor, mais organizado.”

⚖️ Ganho de Informação

Quanto a divisão melhora os dados

Escolhe a maior redução de impureza

“O ganho de informação mede o quanto uma divisão melhora os dados.” “A árvore sempre escolhe a divisão que mais reduz a bagunça.”

🌳 Crescimento

  1. Começa com todos os dados
  2. Escolhe melhor divisão
  3. Divide
  4. Repete
“A árvore começa com todos os dados.” “Depois divide…” “Depois divide de novo…” “E continua até organizar tudo.”

🧠 Quando para?

  • Nó puro
  • Limite de profundidade
  • Poucos dados
“Ela para quando os dados já estão organizados…” “Ou quando chega em um limite…” “Ou quando não faz mais sentido dividir.”

⚙️ Parâmetros

criterion: gini / entropy

max_depth: profundidade

min_samples_split: mínimo para dividir

min_samples_leaf: mínimo na folha

“Aqui temos os parâmetros que controlam a árvore.” “Por exemplo, profundidade, quantidade mínima de dados…” “Esses parâmetros ajudam a evitar erros no modelo.”

⚠️ Overfitting

Modelo decora os dados

Erra dados novos

Solução: limitar a árvore

“Aqui temos o overfitting.” “É quando o modelo aprende até demais, até os erros dos dados.” “Ele funciona bem no treino, mas erra quando aparecem dados novos.”

📉 Underfitting

Modelo simples demais

Não aprende direito

“Já o underfitting é o contrário.” “O modelo é simples demais…” “E não consegue aprender direito.”

📊 KNN vs Decision Tree

KNN → usa distância

Decision Tree → cria regras

“O KNN funciona baseado em distância.” “Já a árvore de decisão funciona criando regras.” “Ou seja, um usa matemática de proximidade…” “E o outro usa lógica.”

🚀 Por que usar?

  • Fácil de entender
  • Rápido
  • Interpretável
“A árvore de decisão é muito usada porque é fácil de entender.” “Ela é rápida…” “E principalmente: dá pra explicar o que ela está fazendo.”

🎤 Resumo

A árvore aprende fazendo perguntas e dividindo os dados até deixá-los organizados.

“Resumindo…” “A árvore aprende fazendo perguntas…” “E vai dividindo os dados…” “Até deixar tudo organizado e chegar numa decisão.”
